# Packet Sniffer (Python)

## Overview

This project is a simple **Packet Sniffer written in Python**.
It captures and analyzes network packets in real time. Packet sniffers are commonly used in **network analysis, troubleshooting, and cybersecurity monitoring**.

The tool helps demonstrate how data travels through a network and how security analysts inspect packets to detect suspicious activity.



## Features

* Captures live network packets
* Displays source and destination IP addresses
* Shows protocol information
* Lightweight and beginner-friendly implementation
* Useful for learning **network security and packet analysis**



## Technologies Used

* Python
* Socket programming
* Basic networking concepts



## How It Works

The script listens to network traffic using raw sockets and extracts important information such as:

* Source IP
* Destination IP
* Protocol
* Packet data

This helps visualize how network communication occurs between devices.




## Educational Purpose

This project is created for **learning purposes only**.
It demonstrates how packet sniffing works and how security professionals analyze network traffic.

Do **not use it on networks you do not own or have permission to monitor**.
